# 文档模板

本文档提供标准的技术文档模板，确保生成的文档格式统一、内容完整。

## 使用说明

### 何时使用模板

- 生成项目总览文档时，使用"项目总览"模板
- 生成架构文档时，使用"整体架构"模板
- 生成模块文档时，使用"核心模块"模板
- 生成 API 文档时，使用"API 参考"模板

### 如何使用模板

1. 根据文档类型选择对应的模板
2. 填充模板中的占位符（如 [项目名称]、[模块 1] 等）
3. 根据实际情况调整章节结构
4. 添加 Mermaid 图表和代码示例
5. 确保所有文本使用中文

### 模板定制

- 可以根据项目特点增删章节
- 保持章节层级清晰（最多 3 级标题）
- 确保每个模板至少包含 1 个 Mermaid 图表
- 代码引用格式统一：`文件路径:行号范围`

## 基础模板

每个文档必须以 YAML frontmatter 开头：


## 分块写入策略

### 何时分块

- 文档预计超过 50 行
- 包含多个大型代码示例
- 包含多个 Mermaid 图表

### 分块方法

**第一次写入**（前半部分）：
```markdown
# [标题]

## 简介
[内容...]

## [章节 1]
[内容...]

### Mermaid 图表
[图表...]

<!-- 待续 -->
```

**第二次写入**（后半部分）：
使用 Edit 工具替换 `<!-- 待续 -->` 标记：
```markdown
## [章节 2]
[内容...]

## [章节 3]
[内容...]

**关键代码引用**：
- [引用列表]
```
